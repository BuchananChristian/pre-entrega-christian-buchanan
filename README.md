Proyecto Integrador: "Coral Reef"
Autor: Christian Buchanan

Descripción:
Este proyecto es una página web, básica, desarrollada como parte del curso de Front-End de Talento-Tech.
La página lleva por título "Coral Reef" y tiene como temática un acuario que vende peces marinos y productos vinculados a la recreación de su hábitat y a su cuidado.
Está estructurada con HTML semántico, respetando la jerarquía de las etiquetas para organizar la arquitectura de su contenido. 
El objetivo es aprender a crear la estructura básica de una página web y prepararla para futuras mejoras con CSS y JavaScript.
